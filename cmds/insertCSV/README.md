# insertCSV
insertCSV takes a csv file and pushes it to an sMAP server.

# Usage
```
Usage of insertCSV:
  -api string
    	API Key with permission to data.
  -in string
    	CSV file that will be inputed into sMAP. (default "in.csv")
  -server string
    	sMAP Server Address
  -uuid string
    	sMAP UUID
```
